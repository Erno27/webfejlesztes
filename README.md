# webfejlesztes
webfejlesztés tárgy beadandója

készítő: Rankasz Ernő

URL: https://erno27.github.io/webfejlesztes/

Téma: Egy elrejtett kisvárosban lévő Luxus Fodrászat weboldala. 

Saját JavaScript kód helye és működése:
A JavaScript kód a "velemenyek.js" fájlban található, és a "velemenyek.html" oldalhoz van csatolva.
A kód két fő funkciót lát el:
1. sikeres kitöltés esetén létrehoz egy új HTML kártyát a megadott névvel, dátummal és csillagokkal
2. Átlagszámítás: Az oldal betöltésekor és minden új vélemény hozzáadásakor megszámolja a csillag emojikat ("⭐"), és kiszámolja az összesített átlagot, amit a címsor alatt ír ki

Használt egyedi fonttípus és helye:
-A projekt a "Noto Serif" betűtípust használja a Google Fonts könyvtárból.
-Helye: A HTML fájl head részében található link címkékkel van importálva, a CSS-ben pedig a "css.css" fájl elején, a globális *-al van beállítva alapértelmezettként.

Felhasznált külső források:
- Google Fonts (Noto Serif betűtípus importálása).
- képek és illusztrációk: Mindegyik felhasznált kép a Gemini által lett létrehozva
-A kód írásában egy alap kezdetleges Html, néhol css fájl generálása mesterséges intelligenciával. illetve a script optimalizálásánál
